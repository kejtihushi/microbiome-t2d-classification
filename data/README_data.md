# Të dhënat

Dataset-i i përdorur është QinJ_2012 (Qin et al., 2012, Nature), i marrë nëpërmjet paketës curatedMetagenomicData (Bioconductor).

Të dhënat nuk ngarkohen në këtë repository. Ato gjenerohen duke ekzekutuar skriptin `scripts/01_download_dataset.R`, i cili i shkarkon dhe i eksporton në këtë folder si `qin2012_abundance.csv` dhe `qin2012_metadata.csv`.

Skriptet pasuese të preprocessing gjenerojnë gjithashtu skedarët e ndërmjetëm (dt_X_train.csv, dt_y_train.csv, dt_feat_train.csv, etj.) në këtë folder.
